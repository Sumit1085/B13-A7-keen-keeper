# Keen Keeper

## 📌 Description
Keen Keeper is a simple contact tracking application that helps users stay connected with their friends. It allows you to record interactions such as calls, texts, and video chats, and visualize your communication habits through an interactive dashboard.

---

## 🛠️ Technologies Used
- React
- JavaScript (ES6+)
- Context API
- React Router
- Recharts
- Tailwind CSS

---

## ✨ Key Features
- 📞 Track friend interactions, including call, text, and video activities  
- 📊 Interactive analytics dashboard with visual charts for communication insights  
- 👥 Clean and responsive interface for managing and viewing friend activity
